# MOST Madrid

Website for **MOST Madrid** — immersive lifestyle & travel experiences in Madrid. Personalized itineraries, exclusive concierge, and cultural curation.

🌐 [most.solutions](https://www.most.solutions/)

## Tech Stack

- [Tailwind CSS v4](https://tailwindcss.com/)
- Static HTML

## Getting Started

```bash
npm install
npm start
```

This runs the Tailwind watcher and a local dev server on port 3000.

### Scripts

| Command          | Description                       |
| ---------------- | --------------------------------- |
| `npm start`      | Dev server + Tailwind watcher     |
| `npm run dev`    | Tailwind CSS watch mode           |
| `npm run build`  | Build & minify CSS for production |
| `npm run serve`  | Start live-server on port 3000    |
| `npm run format` | Format files with Prettier        |
